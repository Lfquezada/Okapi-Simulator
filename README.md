# Okapi-Simulator
A simulator for a population of Okapi and their dependence for external factors such as hunger, predators and hunters.

## Requirements
* Python 3.x
* Numpy
* Scipy
* Matplotlib

## Usage
```
python3 okapisim.py [okapis] [trees] [leopards] [hunters] [cycles to simulate]
```
